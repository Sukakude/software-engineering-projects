# Fullstack Microservices E-commerce Platform

Starbooks is a full-stack online bookstore application built with modern web technologies.
It allows users to browse books, register accounts, verify their emails, make purchases, and manage their profiles.
Admins can manage products, categories, and orders.

---

## Features
### User-related Features
- User Registration & Login
- Email Verification with OTP
- Resend Verification Code
- Secure Authentication (JWT)
- Search & Filter Products
- Add to Cart

### Admin-related Features
- Add / Edit / Delete Books
- Manage Categories
- Manage Users
- View Orders
- Manage Inventory

## My Tech Stack For This Project

### Frontend
- React (Vite)
- Tailwind CSS
- Redux Toolkit
- RTK Query
- React Hook Form

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Nodemailer (Email Service)
- Stripe API

### DevOps
- Docker
- Docker Compose
- Environment Variables
