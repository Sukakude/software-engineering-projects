# Mobile Movie App

This is a **React Native movie app**. It fetches movie data from a public API and displays it in a user-friendly way on both Android and iOS devices.
The goal of this project is to demonstrate how to:
- Set up a React Native project with Expo.
- Fetch data from an API.
- Render movie listings and details.
- Allow users to search for movies.
- Understand navigation and UI components in React Native.

---

## Tools used

- **React Native** – for building the mobile user interface  
- **Expo** – for running and packaging the app  
- **TMDB API** – for movie data  
- **JavaScript / TypeScript**
- **Appwrite** - for data storage (user metrics)
